# StepWise 🧠

A full-stack web app that helps junior developers understand DSA problems step-by-step — without jumping straight to code.

StepWise acts as a **thinking coach**: given a LeetCode-style problem, it breaks down the intuition, approach, and complexity in plain English before showing any solution.

---

## Why StepWise?

Most developers struggle with DSA not because they can't code —
but because they don't know *how to think* about a problem.
StepWise fixes that by teaching the thought process, not just the answer.

---

## Local Setup

### Prerequisites
- Node.js v18+
- MongoDB (local or Atlas)
- Anthropic API Key

### Backend
```bash
cd stepwise-backend
npm install
cp .env.example .env
npm run dev
````

### Frontend

```bash
cd stepwise-frontend
npm install
npm run dev
```
